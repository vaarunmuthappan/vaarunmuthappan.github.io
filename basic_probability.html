Inference: Means making conclusions from data.

Why do we need probability?

  • Formal logic limitations: it describes if something is always or never true, but in real life everything is uncertain and nothing is always true.
  • Using natural language is too subjective, hence we need math to describe it.
  • So we need a way to describe it, we do it with probability.

Sample Space - Omega: Set of all possible outcomes from 1 random experiment.
